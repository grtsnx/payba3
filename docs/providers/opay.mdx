# OPay

OPay is available through the `opay` channel.

```ts
const opay = payba3.use('opay');
```

## Signup and docs

| Resource           | Link                                                |
| ------------------ | --------------------------------------------------- |
| Merchant dashboard | https://merchant.opaycheckout.com                   |
| API docs           | https://doc.opaycheckout.com/payment-authentication |

## Configuration

```ts
const payba3 = createPayba3({
  opay: {
    environment: 'sandbox',
    merchantId: process.env.OPAY_MERCHANT_ID,
    publicKey: process.env.OPAY_PUBLIC_KEY,
    secretKey: process.env.OPAY_SECRET_KEY,
  },
});
```

Environment variables:

```bash
OPAY_ENVIRONMENT=sandbox
OPAY_BASE_URL=
OPAY_MERCHANT_ID=
OPAY_PUBLIC_KEY=
OPAY_SECRET_KEY=
OPAY_LIVE_MERCHANT_ID=
OPAY_LIVE_PUBLIC_KEY=
OPAY_LIVE_SECRET_KEY=
```

## Common methods

| Method                            | Purpose                                       |
| --------------------------------- | --------------------------------------------- |
| `createCashierPayment(input)`     | Create cashier checkout using public-key auth |
| `createPayment(input)`            | Create direct payment using signed auth       |
| `queryPaymentStatus(input)`       | Query payment status                          |
| `cancelPayment(input)`            | Cancel cashier payment                        |
| `createRefund(input)`             | Create refund                                 |
| `queryRefundStatus(input)`        | Query refund status                           |
| `sendOtp(input)`                  | Send OTP for payment flow                     |
| `inputOtp(input)`                 | Submit OTP                                    |
| `inputPin(input)`                 | Submit PIN                                    |
| `verifyCallback(body, signature)` | Verify callback signature                     |

## Cashier checkout

```ts
await payba3.use('opay').createCashierPayment({
  reference: 'order_123',
  amount: {
    total: 250000,
    currency: 'NGN',
  },
  returnUrl: 'https://example.com/return',
  callbackUrl: 'https://example.com/opay/callback',
});
```

## Query payment status

```ts
await payba3.use('opay').queryPaymentStatus({
  reference: 'order_123',
});
```

## Refund

```ts
await payba3.use('opay').createRefund({
  reference: 'refund_123',
  originalReference: 'order_123',
  amount: {
    total: 250000,
    currency: 'NGN',
  },
});
```

Require human approval for live refunds.
